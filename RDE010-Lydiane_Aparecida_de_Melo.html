Há três formas de utilizar o CSS

  1. Inline

    2. Os códigos CSS são definidos dentro de uma tag HTML.
    Afetam somente a tag em que foi defenida.

  2. Incorporado

    3. Os códigos CSS são defenidos no cabeçalho, dentro da tag head.
    Afetam somente o documento em que foi defenido.

  3. Externo

    4. Os códigos CSS são defenidos em um arquivo externo.
    Afetam todos os documentos em que estão linkados ao CSS.

Obs:

O navegador segue essa ordem de prioridade, primeiro aplica os métodos Inline, que vão sobrepor os demais, depois Incorporado e a seguir o Externo.
