Incorporando Multimídias ao site

Multímidias

É uma forma de comunicação que combina diferentes formas de conteúdo, como texto, áudio, imagens, animações ou vídeos em uma única apresentação.

Tag iframe

É usada para incorporar outro documento ao documento HTML, tais como vídeos, outros sites, áudios, imagens, entre outros.

Embutir áudio no site

É bastante utilizado nos sites como de PodCast, onde poderá ter no site áudios explicando sobre um determinado assunto, como de tecnologia, entre outros.
Para poder implementar um áudio no site é necessário abrir uma plataforma com o conteúdo desejado e em compartilhar, escolher a opção Incorporar a mesma disponibiliza para o usuário um código com a tag <iframe> com o tamanho ideal, nesse exemplo usando o soundcloud que permite realizar a hospedagem.

Incorporar vídeos no site

É utilizado por ser bastante prático e que atualmente não compensa ter um servidor dedicado para hospedar os vídeos, por terem durações longas possui um espaço de armazenamento maior.
Para poder implementar um vídeo no site é necessário abrir uma plataforma com o conteúdo desejado e em compartilhar, escolher a opção Incorporar a mesma disponibiliza para o usuário um código com a tag <iframe> com o tamanho ideal, nesse exemplo usando o YouTube que permite realizar de forma gratuita para realizar a hospedagem.

Tag vídeo

É utilizado para poder inserir vídeos que se encontra na máquina localmente ou no servidor do usuário.

Tag audio

É utilizado para poder inserir áudios qu se encontra na máquina localmente ou no servidor do usuário.
Existe uma biblioteca de áudio do YouTube que permite utilizar sem que tenha direitos autorais.

Tag picture

É utilizado para poder agrupar várias imagens.
